Useful Tools

Since I started to learn how to code I’ve slowly built up a list of tools that I use regularly. I’ve put together this list so that others don’t have to go through the hours of searching (though that’s part of the fun right?) as I’ve got to grips with how to build basic web apps using Python.

Learning to Code

  • Code Academy taught me the basics
  • EdX and MITx really got me started
  • And Google of course

Tools & Software

  • Jupyter Notebooks are amazing - why use anything else?
  • Atom for those times when you really need a text editor
  • Anaconda from Continuum makes the process of installing and keeping all the bits of software up to date so much easier
  • Github to manage my codebase

Publishing

  • Github Pages is what this blogs sits on
  • Hexo is a very simple and lightweight blog framework
  • Heroku for anything that can’t be linked to from a blog

Data Wrangling

  • The Pandas Python library is amazingly powerful and easy to use
  • Creating charts with Plotly is simple, and the variety of available plots is huge
  • This one took me a while, but until Juptyer enable sharing dynamic notebooks, the best way I’ve found to display dynamic tables of data is with the venerable Google Sheets plus the Table Awesome add-on.
