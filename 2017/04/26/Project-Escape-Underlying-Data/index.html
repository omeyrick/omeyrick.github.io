Project Escape: Underlying Data

In pulling together my UK “Places to live” map I sourced and cleaned up quite a lot of potentially interesting datasets. Here’s a list of the data that sits behind the map: PlacesData. The schools data in particular is worth looking at in more detail.

National Rail Data

The most useful of all of this is the train times dataset. This was pulled direct from the National Rail Enquiries website in April 2017. The shortest journey time to London was found by searching for the first five trains to leave from 8am each morning, and then taking the shortest route.

You can filter the results by origin and destination, as well as journey time. Hope it’s helpful.
